[Demo] 공유 기능 안내

본 데모에서는 Web Share API로 공유 기능을 구현했습니다. 모바일 기기에서 카카오톡, 문자, 메일 등 설치된 앱으로 바로 공유할 수 있습니다.

실제 서비스에서는 Kakao SDK 연동으로 카카오톡 직접 공유를 구현할 예정입니다.

정적 사이트에서 Kakao SDK 사용 시:

  • 장점 - 무료(일 30만 건), CDN 로드만으로 동작, 서버 불필요, 카카오톡 맞춤 템플릿(썸네일/제목/설명) 지원
  • 단점 - JavaScript 앱 키가 프론트엔드에 노출됨(도메인 제한 설정으로 보완), 커스텀 템플릿은 Kakao 앱 검수 필요, SDK 버전 업데이트 시 수동 대응
